# Modern React To-Do App

A modern and responsive To-Do List application built using React and Vite.

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Responsive UI

## Project Build using: 

- React
- Vite
- CSS3



## How to Run the Project:

# Terminal Commands ---

- npm install
- npm run dev 


*Note : --- "npm run dev" --- will generate a localhost server to open the Application.
